# FXP Usage

Quick examples:

- Create a new resource from template
```
fxp create my-cool-res -t basic-lua
```

- Initialize current folder (writes fxmanifest.lua and folders)
```
fxp init -y
```

- Export a resource to zip
```
fxp export . -o my-cool-res.zip
```

- List templates
```
fxp templates
```
